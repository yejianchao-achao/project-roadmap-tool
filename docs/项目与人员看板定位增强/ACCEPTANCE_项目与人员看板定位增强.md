# 验收阶段（ACCEPTANCE）- 项目与人员看板定位增强

## 验收标准
- 进入进度看板或人员看板时：
  - 今日红线清晰可见，位于项目块之上，颜色为红（`#ff4d4f`）。
  - 本周阴影背景覆盖当周7天范围，半透明蓝（`rgba(24, 144, 255, 0.12)`）。
  - 滚动与缩放时，两者位置同步且不抖动。
  - 修改时间范围为不包含今天或本周时，两者自动隐藏。

## 验收步骤
1. 启动前端服务（端口5173），打开时间轴页面。
2. 验证当前日期位置处有红线；刷新页面保持。
3. 验证当周范围有半透明阴影，产品块下方可见。
4. 切换时间范围到不包含“今天”的范围，红线与阴影自动隐藏。
5. 切回包含今天的范围，红线与阴影恢复显示。

# 产品背景 (Product Context)

## 为什么存在
项目经理和团队需要一种轻量级、可视化的方式来跟踪高层级的项目进度，而无需使用复杂的企业级项目管理软件（如 Jira 或 MS Project）。本工具填补了简单电子表格与复杂 PM 工具之间的空白。

## 解决的问题
1.  **视觉混乱**: 传统的列表无法直观展示时间重叠或资源冲突。本工具使用带有智能布局算法的时间轴视图来解决这一问题。
2.  **上下文切换**: 用户需要按产品线分组查看项目。泳道视图满足了这一需求。
3.  **环境配置困难**: Python/Node 应用通常启动复杂。`start.py` 和 `.command` 脚本自动化了环境设置（venv, npm install 等）。
4.  **数据迁移性**: 对于单用户/小团队的本地工具来说，数据库设置过于繁琐。JSON 存储使得数据易于备份和迁移。

## 用户体验目标
- **即开即用**: 双击即可启动。基本使用无需任何终端知识。
- **响应式设计**: UI 适应窗口大小；表头与内容区域滚动同步。
- **偏好记忆**: 工具会记住您上次查看的产品线和时间轴缩放级别。
- **美观现代**: 采用现代 UI (Ant Design)，通过清晰的颜色编码实现状态的快速识别。

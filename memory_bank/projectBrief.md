# 项目概要 (Project Brief)

## 项目概览
一个基于 Web 的项目路线图可视化管理工具，专为本地管理多条产品线的项目而设计。它提供了一个类似甘特图的时间轴视图，支持泳道展示，涵盖从规划到上线的全生命周期管理。

## 核心目标
- **可视化**: 清晰的时间维度展示，自动处理项目重叠。
- **管理**: 简单易用的项目和产品线增删改查（CRUD）操作。
- **便捷启动**: 为 macOS 用户提供“一键启动”体验，自动处理所有依赖。
- **持久化**: 零数据库配置，使用本地 JSON 文件存储，便于移植和备份。

## 关键特性
- **时间轴视图**: 提供月度和周度网格，包含“今日”和“本周”高亮显示。
- **泳道展示**: 按产品线分组展示项目。
- **状态追踪**: 7 种不同的项目状态，每种状态有独特的颜色标识。
- **智能布局**: 自动行分配算法，防止项目块在视觉上重叠。
- **云部署**: 提供用于将数据备份并部署到远程服务器的脚本。
- **用户个性化**: 自动保存可见产品线和时间轴范围的设置。

## 项目状态
- **当前阶段**: 维护 / 稳定
- **完成度**: 100% 核心功能及计划内的优化已完成。
